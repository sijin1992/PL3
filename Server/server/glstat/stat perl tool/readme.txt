	***古龙数据统计工具**

功能：统计基本信息, 默认方式
使用方式：./glstat -d 20140112 4 
说明：-d 可以指定截止日期，'+'表示截止到今天（./glstat + 4），默认为截止到昨天，
      数字4为截止日期往前推的天数，默认为7天，即 4 days ago，以此表示要统计的时间段


功能：统计等级分布
使用方式：./glstat -l 或者 ./glstat --levels
说明：查看指定时间段内的玩家等级分布，只以登录时记录的等级为统计标准，时间段可以用上面的方式给出

功能：查看单项记录
使用方式：./glstat -s type -u user
说明：查看指定时间段内的某单项记录，单项类型有type给出，比如login表示登录记录，
      -u user指定需要查看的用户名，省略表示查看所有用户，直接-s可查看所有记录的类型，时间段可以用上面的方式给出
